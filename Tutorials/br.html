The Html br tag



The Html br tag is tag or a element that creates line breaks
This tag (element) does not need a end tag
To make one, type <br> where you want a line break

This is a paragraph without a line break:
Some Line One text Some Line Two text

As you can see, they are all toegther, so we add line breaks!

Here are some text WITH line breaks:
Some Line One text
Some Line Two text


See the diffrence?
